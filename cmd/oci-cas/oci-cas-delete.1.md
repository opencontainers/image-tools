# NAME

oci-cas-get \- Remove a blob from the store

# SYNOPSIS

**oci-cas delete** [OPTIONS] PATH DIGEST

# DESCRIPTION

`oci-cas delete` removes the blob referenced by `DIGEST` from the store at `PATH`.

# OPTIONS

**--help**
  Print usage statement
