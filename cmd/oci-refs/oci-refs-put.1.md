# NAME

oci-refs-put \- Write a reference to the store

# SYNOPSIS

**oci-refs put** [OPTIONS] PATH NAME

# DESCRIPTION

`oci-refs put` reads descriptor JSON from standard input and writes it to the store at `PATH` as `NAME`.

# OPTIONS

**--help**
  Print usage statement
