# NAME

oci-refs-delete \- Remove a reference from the store

# SYNOPSIS

**oci-refs delete** [OPTIONS] PATH NAME

# DESCRIPTION

`oci-refs delete` removes reference `NAME` from the store at `PATH`.

# OPTIONS

**--help**
  Print usage statement
